Welcome to Draw contest!

Here you can enter one of the available room from the list, or create a new one!

Draw content is a real-time draw-and-guess game with simple rules!

  • One player tries to draw an image of something
  • The others try to guess what it is
  • If a player guesses the correct word, both that player and the drawer gets awarded with one point each.
  • If time runs out, a new drawer is randomly chosen and no point are awarded
